# Git options

https://www.reddit.com/r/programming/comments/1bgyhl1/popular_git_config_options/
https://jvns.ca/blog/2024/02/16/popular-git-config-options/


Nice defaults:
```gitconfig
[alias]
	lg = log --color --graph --pretty=format:'%Cred%h%Creset -%C(yellow)%d%Creset %s %Cgreen(%cr) %C(bold blue)<%an>%Creset' --abbrev-commit
[filter "lfs"]
	clean = git-lfs clean -- %f
	smudge = git-lfs smudge -- %f
	process = git-lfs filter-process
	required = true
[init]
	defaultBranch = main
[commit]
	verbose = true
[core]
	pager = delta
[rebase]
	autosquash = true
[rerere]
	enabled = true
[diff]
	algorithm = histogram
[interactive]
	diffFilter = delta --color-only
[delta]
	line-numbers = true
```

Signing setup (don't need KeepassXC specifically)
https://code.mendhak.com/keepassxc-sign-git-commit-with-ssh/

```gitconfig
[commit]
	gpgsign = true
[gpg]
	format = ssh
[gpg "ssh"]
	allowedSignersFile = /absolute/path/to/ssh/allowed/signers/file
```
